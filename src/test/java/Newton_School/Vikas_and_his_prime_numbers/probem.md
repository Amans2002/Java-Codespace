# [Problem: Vikas and his prime numbers](https://my.newtonschool.co/playground/code/rgijz3bxan8s)

Ram admires Vikas, a brilliant mathematician, and wants to become skilled in number theory as well. Vikas suggests that Ram should practice identifying prime numbers to strengthen his skills. Ram is determined to follow Vikas' advice and improve his understanding of prime numbers. <br>
Your task is to help Ram by writing a program that can efficiently find all prime numbers between 1 and a given number, n.

## Input

The first and the only line of the input contains an integer n, representing the number upto where you have to print all the prime numbers starting from 1.

## Constraints

**1 ≤ n ≤ 10<sup>6</sup>**

## Output

Output contains all prime numbers space separated between 1 and n(inclusive)

## Sample

- Input:
```
13
```

- Output:
```
2 3 5 7 11 13
```
