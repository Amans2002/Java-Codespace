# [Problem: Large Factorial](https://www.interviewbit.com/problems/large-factorial/)

Given a number **A**. Find the fatorial of the number.

## Input

First and only argument is the integer **A**.

## Constraints

**1 ≤ A ≤ 100**

## Output

Return a string, the factorial of **A**.

## Sample

- Input:
```
2
3
```

- Output:
```
2
6
```

##Explanation
- For case 1: **2! = 2** <br>
- For case 2: **3! = 6**
